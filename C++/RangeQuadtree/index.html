Range Quadtree
General information can be found on wikipedia.

Features of my implementation:

• Quadtree is subdivided only if the whole rectange fits within the boundary od subtree quadrant. This subdivision continues to maximum possible depth. This implies that rectangles can be stored in non-leaf trees.

• Arbitrary limit for maximum quadtree depth.

• Dedicated implementation of point relocation. Same principle as in Point Quadtree. The speedup is not as significant however due to different way of storing the elements.

• Both range and point querying is possible.
